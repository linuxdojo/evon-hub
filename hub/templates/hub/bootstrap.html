Evon Bootsrap is a run-once shell script that persistently connects a Linux system to your overlay network. Evon Bootstrap can also be run using Docker. The following Linux distributions are direclty supported:

  • AlmaLinux
  • Alpine
  • Amazon Linux 2
  • Arch Linux
  • CentOS 7 or higher
  • Debian 9 or higher
  • Fedora Linux
  • Rocky Linux
  • Ubuntu 18.04 or higher
  • OpenSUSE 15 or higher
  • Any modern Linux distribution using Docker
Note: The API Token associated with the deployer user is required when both downloading and installing Bootstrap, and can be managed here. The below CLI invocations include this token for your convenience.
Download Bootstrap
Direct download:
Direct download link: bootstrap.sh

Download using curl:
Run the following command on your server: 
curl -s 'https://{{ account_domain }}/api/bootstrap/download' -H 'Authorization: Token {{ deploy_token }}' > bootstrap.sh; chmod +x bootstrap.sh
Download using wget:
Run the following command on your server: 
wget --header 'Authorization: Token {{ deploy_token }}' 'https://{{ account_domain }}/api/bootstrap/download' -O bootstrap.sh; chmod +x bootstrap.sh
Using Docker:
Visit https://hub.docker.com/r/linuxdojo/evon-bootstrap for detail about using the Dockerised Boostrap. See below for invocation details.
Run Bootstrap
Using bootstrap.sh:
Run the following command as root on your server: 
EVON_DEPLOY_KEY={{ deploy_token }} ./bootstrap.sh --install
Note: Your server needs to be able to connect to {{ account_domain }} on TCP/443 either directly or via a https proxy for Bootstrap run successfully. For help regarding proxy setup, uninstallation and other options, run command: 
./bootstrap.sh --help
Using Docker:
The Dockerised Bootstrap will require the following to environment variables to be set, as referenced on the Evon Bootstrap Docker Hub page: 
ACCOUNT_DOMAIN={{ account_domain }}
EVON_DEPLOY_KEY={{ deploy_token }}
