Evon Bootsrap is a run-once shell script or Docker container that persistently connects a Linux or Windows system to your overlay network.

The following Windows systems are supported:

  • Windows 7 with Service Pack 1 or higher + Windows Management Framework 5.1
  • Windows 10 version 1607 or higher
  • Windows Server 2008 R2 with Service Pack 1 or higher + Windows Management Framework 5.1
  • Windows Server 2016 or higher
The following Linux distributions are supported:

  • AlmaLinux
  • Alpine
  • Amazon Linux 2
  • Arch Linux
  • CentOS 7 or higher
  • Debian 9 or higher
  • Fedora Linux
  • Red Hat Enterprise Linux
  • Rocky Linux
  • Ubuntu 18.04 or higher
  • OpenSUSE 15 or higher
  • Any Linux distribution using Docker
Note: The API Token associated with the deployer user is required when both downloading and installing Bootstrap, and can be managed here. The below CLI invocations include this token for your convenience.
Download and Install
  • Linux
  • Windows
  • Docker
For Linux servers, download and install Bootstrap using the process described below.

Download bootstrap:
Direct download:

    Direct download link: bootstrap.sh

Using curl:
curl -s 'https://{{ account_domain }}/api/bootstrap/linux' -H 'Authorization: Token {{ deploy_token }}' > bootstrap.sh; chmod +x bootstrap.sh
Using wget:
wget --header 'Authorization: Token {{ deploy_token }}' 'https://{{ account_domain }}/api/bootstrap/linux' -O bootstrap.sh; chmod +x bootstrap.sh
Install Bootstrap:
Run the following command as root on your server: 
EVON_DEPLOY_KEY={{ deploy_token }} ./bootstrap.sh --install
Note: Your server needs to be able to connect to {{ account_domain }} on TCP/443 either directly or via a HTTPS proxy for Bootstrap to run successfully. For help regarding proxy setup, uninstallation and other options, run command: 
./bootstrap.sh --help
For Windows servers, download and install Bootstrap using the process described below.

Download bootstrap:
Direct download:

    Direct download link: bootstrap.ps1

Using PowerShell:
Invoke-WebRequest -Uri 'https://{{ account_domain }}/api/bootstrap/windows' -Headers @{"Authorization"="Token {{ deploy_token }}"} -Outfile bootstrap.ps1
Install Bootstrap:
Run the following PowerShell command on your server as Administrator: 
.\bootstrap.ps1 -Apikey {{ deploy_token }}
Note: Your server needs to be able to connect to {{ account_domain }} on TCP/443 either directly or via a HTTPS proxy for Bootstrap to run successfully. For help regarding proxy setup, uninstallation and other options, run command: 
.\bootstrap.ps1 -Help
Docker Bootstrap
As an alternative to downloading and installing Bootstrap directly onto your Linux distribution, Bootstrap can be run entirely within a Docker container.

Visit https://hub.docker.com/r/linuxdojo/evon-bootstrap for detail about using the Dockerised Bootstrap. The Docker version of Bootstrap will require the following to environment variables to be set: 
ACCOUNT_DOMAIN={{ account_domain }}
EVON_DEPLOY_KEY={{ deploy_token }}
